# Halo Spiro Ribbons Studio

An interactive browser studio for building Halo-style spiro ribbon artwork with Leva controls, Canvas 2D rendering, image upload, animation controls, and PNG/SVG export.

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Deploy To Vercel

1. Import this repository in Vercel.
2. Use the default Next.js settings.
3. Deploy.

No private package registry or environment variables are required.

## Features

- Leva side controls for ribbon, image, animation, and export settings
- Figma-inspired spiro ribbon presets
- Upload, replace, and remove raster images
- Image-behind, sampled-color, and clipped-image render modes
- Play/pause animation, speed, phase, reveal, and pulse controls
- PNG export for all states
- SVG export when no uploaded raster image is active
- Copy settings JSON and shareable URL config
